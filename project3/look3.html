COLLEGE WEB APPLICATION AND WEBSITE FOR STUDENT INFORMATION

ROLE: DEVELOPER

Environment : Java, Spring Boot, Hibernate, HTML, CSS, MySQL

Description :To provide a user-friendly interference to students to get official information, notice, timetable, etc. by the college through an application development. A web application connected to cloud storage and gets updated with every hour during college office time which will be controlled by Dean and the Principal.

RESPONSIBILITIES :

  • Backend development and management using Spring Boot and MySQL database
  • Created a database as per ER diagram & SRS documentation.
  • Used Hibernate for object-relational mapping and persistence and used spring framework for development
