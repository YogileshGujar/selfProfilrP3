ONLINE CAR PARKING SYSTEM

ROLE: DEVELOPER

Environment : Java, Spring Boot, Hibernate, HTML, CSS, MySQL

Description : Online Car Parking System is a Java Spring Bootbased portal designed to manage parking slot vacancy, charges and all the records to provide a secure and hassle-free parking experience. Users of this website will need to register themselves with car details which help to track checkin and check-out time of the car. Admin can manage parking places, slots, customers, staff and records.

RESPONSIBILITIES :

  • Development of the project from scratch like planning, execution, testing, modifying.
  • Created a database as per ER diagram & SRS documentation.
  • Used Hibernate for object-relational mapping and persistence and used spring framework for development.
